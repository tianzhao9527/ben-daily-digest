## 可视化配置编辑

打开设置面板后：

### 类目管理
- 点击 ✎ 编辑现有类目
- 点击 × 删除类目  
- 点击 "+ 添加新类目" 创建
- 完成后点击 "导出类目JSON"

### KPI指标管理
- 在"数据源"选项卡管理KPI
- 添加/编辑/删除指标
- 导出KPI JSON

### 重要性系数
- 直接在输入框修改数值
- 导出权重JSON

### 导出完整配置
- 在"配置"选项卡查看完整配置
- 点击"下载配置文件"获取 digest_config_v15.json
- 将文件上传到 GitHub 仓库替换原文件

## KPI 指标分组

| 分组 | 指标 | 可折叠 |
|------|------|--------|
| 股票指数 | 标普500, 纳斯达克, 道琼斯, VIX, 日经225 | ✅ |
| 利率债券 | 美债10Y/2Y, 利差, 联邦基金利率 | ✅ |
| 外汇汇率 | USD/CNY, USD/JPY, EUR/USD, 美元指数 | ✅ |
| 能源商品 | WTI, 布伦特, 天然气 | ✅ |
| 金属价格 | 铜, 铝, 锌, 镍 | ✅ |

点击分组标题可折叠/展开该分组。
