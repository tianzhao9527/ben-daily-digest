## 移动端使用

在手机上访问时，底部会显示4个导航按钮：

| 按钮 | 功能 |
|------|------|
| 📰 新闻 | 主内容区 |
| ☰ 导航 | 左侧分类导航 |
| 📊 指标 | 右侧KPI数据 |
| ⚙ 设置 | 配置面板 |

## 可视化配置编辑

打开设置面板：

### 类目管理
- 点击 ✎ 编辑类目（修改名称、关键词）
- 点击 × 删除类目
- 点击 "+ 添加新类目" 创建新类目
- 点击 "导出类目JSON" 下载配置

### KPI指标管理
- 在"数据源"选项卡管理指标
- 添加/编辑/删除FRED指标
- 导出KPI JSON

### 导出完整配置
1. 编辑完成后，切换到"配置"选项卡
2. 点击"下载配置文件"
3. 将 `digest_config_v15.json` 上传到GitHub替换原文件
4. T+1 生效
