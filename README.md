# Ben Daily Digest v3.0

智能新闻简报生成器 - 金属贸易/地缘政治/AI算力专注版

## v3.0 新功能

### 📊 KPI增强
- **MA5/MA20均线** - 橙色MA5，紫色MA20
- **支撑/阻力位** - 基于20日高低点
- **增强迷你图** - 显示多条均线

### 🎯 市场情绪仪表盘（Apple风格）
- 渐变圆弧进度条
- 精致指针动画
- 看多/中性/看空圆形徽章

### 📅 月日历选择器
- 简约月历视图
- 有数据的日期带蓝点标记
- 左右翻月导航

### 📰 新增类目
- **AI / 大模型** - GPT、LLM、应用
- **新能源汽车** - 电池、充电、智驾
- **宇宙探索** - 航天、卫星、深空

### ⚙️ 配置优化
- 每板块稳定10条新闻
- 原始抓取增至30条
- 总计11个新闻板块
