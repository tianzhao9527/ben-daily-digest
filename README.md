# Ben Daily Digest v3.0

智能新闻简报生成器 - 金属贸易/地缘政治/AI算力专注版

## v3.0 新功能

### 右侧数据面板新增

1. **市场情绪仪表盘**
   - SVG仪表盘显示整体情绪
   - 情绪指数 0-1（0看空，1看多）
   - 看多/中性/看空分布统计

2. **热门关键词云**
   - 从所有板块提取关键词
   - 频率越高字体越大
   - 最多显示12个关键词

3. **板块热度图**
   - 8个板块新闻数量可视化
   - 颜色深浅表示热度

### 板块标题增强

- 每个板块显示情感标签（看多/中性/看空 + 百分比）
- 显示LLM提取的关键词标签

### GitHub配置同步

- 设置 → 同步 tab
- 支持完整URL或 owner/repo 格式
- 直接通过API推送配置更改

## 设置面板

| Tab | 功能 |
|-----|------|
| 通用 | 基本信息、深色模式 |
| 类目管理 | 可视化编辑新闻板块 |
| 重要性系数 | 调整新闻权重 |
| 算法 | 查看LLM配置 |
| 数据源 | 编辑KPI指标 |
| 配置 | 导出完整配置JSON |
| **同步** | GitHub配置同步（新增）|

## GitHub Secrets

| Secret | 必需 | 说明 |
|--------|------|------|
| DEEPSEEK_API_KEY | ✓ | DeepSeek API密钥 |
| QWEN_API_KEY | 可选 | 通义千问备用 |
| OPENAI_API_KEY | 可选 | OpenAI备用 |
| GNEWS_API_KEY | ✓ | GNews新闻API |
| FRED_API_KEY | ✓ | FRED经济数据API |
