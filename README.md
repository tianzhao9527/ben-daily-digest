# Ben's Daily Brief v2.7

苹果风格新闻简报系统，支持历史回溯和移动端访问。

## v2.7 更新

| 功能 | 状态 |
|------|------|
| **添加按钮样式** | ✅ 蓝色高亮，更醒目 |
| **表单位置优化** | ✅ 新增时显示在按钮下方 |
| **max_tokens增加** | ✅ 3000→4500，减少截断 |
| **JSON解析增强** | ✅ 支持截断修复 |
| **历史简报回溯** | ✅ 点击日期查看历史 |

## 本次修复

### 1. 添加按钮样式
- 从灰色虚线改为蓝色虚线
- hover时变为蓝色实心
- 更容易看到和点击

### 2. 表单位置
- 点击"+ 添加"后，表单显示在按钮**下方**
- 自动滚动到表单位置
- 编辑时表单显示在列表上方

### 3. JSON解析问题
- 增加 `max_tokens` 从 3000 到 4500
- 支持修复被截断的JSON
- 正确处理markdown代码块包裹的JSON

## 重要提示

**请确保部署最新代码！** 如果日志仍显示 "non-JSON" 错误：
1. 下载最新的zip包
2. 完全替换GitHub仓库中的所有文件
3. 重新运行 GitHub Action

## 历史简报功能

点击页面顶部的日期可查看历史简报（自动存档90天）。

## 移动端使用

底部显示4个导航按钮：📰新闻 / ☰导航 / 📊指标 / ⚙设置

## 多LLM API 配置

| 优先级 | 提供商 | Secret名称 |
|--------|--------|------------|
